JSX

Please open the console with F12

Ex1

Do not render unknown element.

Ex2

Do not render an element starting with uppercase. It will be considered as React component.

Ex3

Do not use attribute class, use className instead.

Ex4

Better should use parenthesis for wrapping multiline JSX. Avoid automatic column insertion problems.

Ex5

Always use an enclosing tag, at least a React.Fragment

Ex6

Javascript expression can be used inside TextElement.

Ex7

Javascript expression can be used inside attribute.

Ex8

Self closing tag are permitted.

Ex9

Dot notation for element is permitted, but not [] notation.

Ex10

Spread attribute.

Notes

  • Be sure to run the Chrome browser with options --allow-file-access-from-files
  • JSX create React.createElement() javascript code.
  • Question: how we do a comment inside a JSX?
