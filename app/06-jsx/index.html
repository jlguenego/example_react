JSX

Please open the console with F12

Notes

  • Be sure to run the Chrome browser with options --allow-file-access-from-files
  • JSX create React.createElement() javascript code.

Rules in JSX.

  1. Do not render unknown element.
  2. Do not render an element starting with uppercase. It will be considered as React component.
  3. Do not use attribute class, use className instead.
  4. Better should use parenthesis for wrapping multiline JSX. Avoid automatic column insertion problems.
  5. Always use an enclosing tag, at least a React.Fragment
  6. Javascript expression can be used inside TextElement.
  7. Javascript expression can be used inside attribute.
  8. Self closing tag are permitted.
  9. Dot notation for element is permitted, but not [] notation.
  10. Spread attribute
